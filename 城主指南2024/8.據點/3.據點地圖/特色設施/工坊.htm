工坊 Workshop
5級據點設施

先決條件：無
空間：寬敞
僱員：3
指令：製造

工坊是充滿創意的地方，許多有用的東西都能在這裡製造。

工匠工具 Artisans』s Tools ：工坊被加入據點時，其內有六種工匠工具Artisans』s Tools可以投入使用。從下述列表中進行選擇：

  • 木工工具 Carpenter』s Tools
  • 鞋匠工具 Cobbler』s Tools
  • 玻璃匠工具 Glassblower』s Tools
  • 珠寶匠工具 Jeweler』s Tools
  • 皮革匠工具 Leatherworker』s Tools
  • 畫家工具 Pianter』s Tools
  • 陶藝工具 Potter』s Tools
  • 石匠工具 Mason』s Tools
  • 修補工具 Tinker』s Tools
  • 織布工具 Weaver』s Tools
  • 木雕工具 Woodcarver』s Tools

製造選項 Craft Options ：當你對此設施下達製造指令時，從以下選項中選擇一項：

  • 製造：冒險裝備 Craft：Adventuring Gear ：設施僱員製造任意能用工坊內已有的工匠工具（見上文）製造的物品，使用玩家手冊中的相關規則。
  • 製造：魔法物品（器具） Craft：Magic Item（Implement） ：如果你的等級達到或高於 9 級，你可以委託此設施的僱員製造一件由你選擇的普通或非普通魔法物品，該物品必須來自第 7 章的器具 Implement表格。此設施擁有製造該物品所需的工具，而其僱員擁有該工具的熟練以及奧秘技能的熟練。參考第 7 章「製造魔法物品」一節來決定製造該物品所需的時間與金錢。如果該物品允許使用者用它施展任何法術，你必須親自製造它（設施僱員可以協助你），並且在製造該物品期間，你每天都必須準備所有該物品可以施展的法術。

靈感之源 Source of Inspiration ：當你在工坊內完成一次完整的短休后，你獲得英雄激勵 Heroic Inspiration。你在完成一次長休前不能再次獲得此增益。

擴大設施 Enlarging the Facility ：你可以花費 2000 GP ，將工坊擴大為龐大設施。這樣做之後，工坊獲得額外兩位僱員以及額外三種工匠工具（從上述列表中選擇）。
