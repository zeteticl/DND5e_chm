冒險契機Adventure Hooks

你可以根據以下內容改編《施特拉德的詛咒》中的冒險契機。

  • 求助者 Plea for Help。 阿利格爾 Arrigal 是其中一個苦心人，他的衣著和口音都不是特別怪異。當冒險家們在史頓襄的內陸山谷時，他會來找他們，也許是在席爾本 Silbern 或沃勒班克 Wollebank 。那裡的人知道這些苦心族人來自外域山谷。冒險契機在其他方面沒有改變。
  • 神秘訪客 Mysterious Visitors 。 這裡描述的場景發生在內陸山谷的沃勒班克 Wollebank 。這些角色不是和莫文夫人 Lady Morwen 一起吃飯，而是和沃勒班克市長戴根哈德 Deigenhard 一起吃飯。 Deigenhard 很熟悉這些苦心族人，他認為他們只不過是強盜，所以他要求玩家角色讓他們繼續前進。
    如果探險者們跟隨這些苦心族前往外域的山谷，他們會沿著陰鬱的道路 Somber Road 向西北行進，穿過霍夫薩德爾隘口 Hofsaddel 。
  • 霧中的狼人 Werewolves in the Mist 。 狼人正從松柏森 Somberwald 涌向內陸山谷，威脅著那裡的牧羊人和農民。冒險者是基於他們與依尼翠當地組織的關係而獲得資訊 ( 而不是《 D&D 冒險者聯盟》中的角色派系 ) 。如果角色與任何這些組織有密切聯繫，他們就可以獲得與相應玩家陣營相關的資訊和利益：
    用 parish-blades 代替 Harpers ，特別是 Stensian 分支，隨意稱為道路觀察者。
    用 Saint Raban 的騎士審判官代替鐵手勳章 Order of the Gauntlet, 。他們的名字來自於傳說中的偵探和狼人殺手，這個凈化派的組織在經典意義上充當了審判者的角色 不是屠殺平民，而是調查神秘事件和秘密。
    他們是偵探、調查人員，鉆研最好不要觸及的謎。
    與其使用翡翠飛地 Emerald Enclave ，不如使用一組蒼鷺的大法師，包括一些德魯伊。
    使用凱錫革之箭 Quiver of Kessig 代替領主聯盟 Lords』 Alliance ，他們擅長獵殺狼人。
    維恩 Eravien ( 凱錫革人 Kessiger ，而不是《施特拉德的詛咒》中的半精靈 ) 告訴探險者們狼人是蒙卓軍團的一個分支，他們使用黑暗形式的血魔法。
    用一個與哈文居爾市長赫扎格 (Lord Mayor Herzag of Havengul) 有聯繫的犯罪集團代替散塔林，赫扎格是一名腐敗商人，爲了個人利益允許非法地下組織蓬勃發展。
    角色們在追捕狼人的過程中穿過了霍夫薩德爾 Hofsaddel 山口。沒有霧吞沒他們，但他們出現在了外域山谷。
    Creeping Fog 。因為沒有迷霧將角色運送到史頓襄的外域山谷，所以如果你希望在依尼翠之中執行《施特拉德的詛咒》，這個引子就不適用了。


生物和遭遇 Creatures and Encounters

使用這些指導方針來適應整個冒險的生物和遭遇。
巴洛克偵察兵Barovian Scouts 。 當玩家角色第一次遇到這些陰鬱的獵人時，他們的團隊是由一個名叫 Paulin 的人領導的 ( 玩家可能會在最初的 Innistrad 塊上的幾張卡片上認出他 ) 。他給他們留下了幾句至理名言 :「 把你們的死人埋得深一點。野豬在屍體還暖和的時候最餓了。 」( 在你的自由裁量權下，你也可以在隨機遭遇中用相同數量的巨野豬 giant boar 替換狂戰士 berserkers 。 )
布魯托Bluto 。在茂威湖被發現的布魯托在 Emrakul 接近的影響下發瘋了。他獻身於一種他認為生活在湖底的水生生物 類似於涅非利亞海岸邪教崇拜的吉特羅格 Gitrog 怪物或烏賊狀的克拉瑪。布魯托的脖子上有鰓狀的皮瓣。根據你的判斷，你可以介紹他與崇拜的生物的相遇，可能是一個巨大的鱷魚。
巴拉托克山的瘋狂法師Mad Mage of Mount Baratok。 你可以使用《施特拉德的詛咒》中所寫的魔鄧肯。或者，在《暗影依尼翠》的故事情節中，這個瘋狂的旅法師可能是傑斯貝連 Jace Beleren 。為傑斯 Jace 建立一個不同的法術列表，專注於改變思維和幻覺魔法。
Mongrelfolk 。 這些畸形生物的統計數據是相同的，但在外觀上，他們很像在 Emrakul 的影響下扭曲變形的墮落生物。比起爪子，它們可能會使用觸鬚 ( 造成鈍擊傷害 ) 或額外的嘴 ( 造成穿刺傷害 ) 進行攻擊。沒有魔法可以把 Mongrelfolk 恢復到原來的狀態。
吸血鬼衍體Vampire Spawn 。 當探險者遇到吸血鬼怪物時，使用這個檔案中的吸血鬼新生兒統計，而不是怪物手冊中的吸血鬼衍體的統計數據。冒險家們第一次遇到兩個吸血鬼的後代時，他們遇到了茂威莊園臭名昭著的雙胞胎，兩個年輕的女孩名叫魯比 Ruby 和卡邁恩 Carmine 。這兩個人喜歡假裝成無助的年輕女孩，迷路了，孤身一人，以便接近他們的獵物。
渡鴉Wereravens 。 使用 wereraven 統計在冒險，除了喙攻擊不能傳播變狼人 ( 因為女巫的看守羽毛不是變狼人 ) 。 wereraven 也有德魯伊 druid 在怪物手冊中顯示的施法能力。
