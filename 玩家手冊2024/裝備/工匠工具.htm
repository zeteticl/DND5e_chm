工匠工具 Artisan's Tools

工匠工具的主要用途為製造物品或輔助交易貿易。使用這些工具需要擁有對應的獨立熟練。

鍊金工具 Alchemist's Supplies (50GP)

屬性：智力

重量：8磅

操作：辨析一種物質（DC15），或生起一場火（DC15）

製造：強酸、熾火膠、材料包、燃油、紙張、香水

釀酒工具 Brewer's Supplies (20GP)

屬性：智力

重量：9磅

操作：檢測飲品是否下毒（DC15），或辨識酒精（DC10）

製造：抗毒劑

書法工具 Calligrapher's Supplies (10GP)

屬性：敏捷

重量：5磅

操作：以華麗的字跡撰寫文字防止造假（DC15）

製造：墨水、法術卷軸

木匠工具 Carpenter's Tools (8GP)

屬性：力量

重量：6磅

操作：封死或撬開一扇門或一個容器（DC20）

製造：短棒、巨棒、長棍、木桶、箱子、梯子、長桿、攜帶型攻城錘、火把

製圖工具 Cartographer's Tools (15GP)

屬性：感知

重量：6磅

操作：為一小片區域繪製地圖（DC15）

製造：地圖

鞋匠工具 Cobbler's Tools (5GP)

屬性：敏捷

重量：5磅

操作：改造足具來為穿戴者的下一次敏捷（特技）檢定提供優勢（DC10）

製造：攀爬工具

廚師工具 Cook's Utensils (1GP)

屬性：感知

重量：8磅

操作：改善食物風味（DC10），檢查食物是否腐壞或有毒（DC15）

製造：口糧

玻璃匠工具 Glassblower's Tools (30GP)

屬性：智力

重量：5磅

操作：判斷一個玻璃物件在過去24小時內盛過什麼內容物（DC15）

製造：玻璃瓶、放大鏡、望遠鏡、小瓶

珠寶匠工具 Jeweler's Tools (25GP)

屬性：智力

重量：2磅

操作：判斷珠寶價值（DC15）

製造：奧術法器、聖徽

皮匠工具 Leatherworker's Tools (5GP)

屬性：敏捷

重量：5磅

操作：對一個皮製品進行圖案設計（DC10）

製造：投石索、鞭子、獸皮甲、皮甲、鑲釘皮甲、揹包、弩矢匣、地圖或卷軸匣、羊皮紙、小包、箭袋、水袋

石匠工具 Mason's Tools (10GP)

屬性：力量

重量：8磅

操作：在石頭上鑿出一個符號標誌或一個洞（DC10）

製造：滑輪組

畫家工具 Painter's Supplies (10GP)

屬性：感知

重量：5磅

操作：畫出你所見過的某個事物的可辨影象（DC10）

製造：德魯伊法器、聖徽

陶匠工具 Potter's Tools (10GP)

屬性：智力

重量：3磅

操作：判斷一個陶瓷物件在過去24小時內盛過什麼內容物（DC15）

製造：壺、燈

鐵匠工具 Smith's Tools (20GP)

屬性：力量

重量：8磅

操作：撬開一扇門或容器（DC20）

製造：任意近戰武器（除了短棒，巨棒，長棍和鞭子）、中甲（除了獸皮甲）、重甲、滾珠、吊桶、鐵蒺藜、鏈條、撬棍、槍械子彈、爪鉤、鐵壺、鐵釘、投石索子彈

修補工具 Tinker's Tools (50GP)

屬性：敏捷

製造：火銃、手銃、鈴鐺、牛眼提燈、扁瓶、附蓋提燈、捕獵陷阱、鎖、鐐銬、鏡子、鏟子、訊號笛、火絨盒

織布工具 Weaver's Tools (1GP)
