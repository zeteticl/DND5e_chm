其他工具 Other Tools

這些工具能夠輔助冒險和其他事情。

易容工具 Disguise Kit (25GP)

屬性：魅力

重量：3磅

操作：化妝（DC10）

製造：戲服

文書偽造工具 Forgery Kit (15GP)

屬性：敏捷

重量：5磅

操作：模仿其他人的筆跡，至多10個單詞（DC15），或偽造一份火漆（DC20）

賭具（多種） Gaming Set (Varies)

屬性：感知

重量：—

操作：判斷某人是否作弊（DC10），或贏取遊戲（DC20）

變體：骰子（1SP）、龍棋（1GP）、紙牌（5SP）、三龍牌（1GP）

草藥工具 Herbalism Kit (5GP)

屬性：智力

重量：3磅

操作：辨認植物（DC10）

製造：抗毒劑、蠟燭、醫療包、治療藥水

樂器（多類） Musical Instrument

屬性：魅力

重量：多類

操作：演奏一個熟知的曲調（DC10），或是演奏一首即興的樂曲（DC15）

變體：風笛（30GP，6磅）、鼓（6GP，3磅）、揚琴（25GP，10磅）、長笛（2GP，1磅）、號角（3GP，2磅）、魯特琴（35GP，2磅）、里拉琴（30GP，2磅）、排簫（12GP，2磅）、蘆笛（2GP，1磅）、提琴（30GP，1磅）

領航工具 Navigator's Tools (25GP)

屬性：感知

重量：2磅

操作：計劃路線（DC10），或通過觀星判斷位置（DC15）

毒藥工具 Poisoner's Kit (50GP)

屬性：智力

重量：2磅

操作：偵測有毒物件（DC10）

製造：基礎毒藥

盜賊工具 Thieves' Tools (25GP)

屬性：敏捷

重量：1磅

操作：撬鎖（DC15），或解除陷阱（DC15）
