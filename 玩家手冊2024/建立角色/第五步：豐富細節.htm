第五步：豐富細節 Step5:Fill In Details

現在來填寫角色卡的其他部分。

記錄職業特性 Record Class Features
檢視第三章里你職業的特性表，將1級的特性抄下來。這些職業特性的描述也在第三章。
某些職業特性提供了選擇。確定你完整讀過所有的特性，並作出了所有被提供的選擇。

填上數字 Fill In Numbers
把這些「數字」填到你的角色卡上：
豁免檢定Saving Throws。對於你具有熟練的豁免檢定，將你的熟練加值加到相應的屬性調整值上，記錄總值。某些玩家喜歡把他們不熟練的豁免檢定調整值也記錄下來，那就僅是相應的屬性調整值。
技能Skills。對於你具有熟練的技能，將你的熟練加值加到與該項技能聯用的屬性調整值上，記錄總值。你或許想將你不熟練的技能調整值也記錄下來，那就僅是聯用的屬性調整值。
被動察覺Passive Perception。有時候你的DM會不要求你進行感知（察覺）檢定就判定你的角色是否注意到某些東西，此時DM將使用你的被動察覺作為替代。被動察覺這個數值反映了當你沒有主動尋找某些東西時的，一種通常的、對周圍環境的警覺。使用下列公式確定你的被動察覺值：

被動察覺 = 10 + 感知（察覺）檢定的調整值

將適用於你感知（察覺）檢定的所有調整值都算上。比如，如果你的角色感知15且熟練於察覺技能，你的被動察覺就是14（10+感知調整值2+熟練2）。
生命值 Hit Points 。你的職業和體質調整值確定了你在1級時的生命值上限，如各職業1級生命值表所述。

各職業1級生命值 Level 1 Hit Points by Class
職業生命值上限
野蠻人12+體質調整值
戰士、聖武士或遊俠10+體質調整值
吟遊詩人、牧師、德魯伊、武僧、遊蕩者或魔契師8+體質調整值
術士或法師6+體質調整值

角色卡預留了空間，用於在你受到傷害時記錄目前生命值，以及記錄你可能獲得的任何臨時生命值。卡上也有空間用於追蹤死亡豁免檢定。
生命值骰Hit Point Dice 。你的職業描述中說明了你的角色的生命值骰（或簡稱生命骰）型別，將它寫在你的角色卡上。在1級時，你的角色有1個生命骰。你可以在短休時消耗生命骰來恢復生命值。你的角色卡上還有預留的空間，可以記錄你已經消耗的生命骰數量。
先攻Initiative 。在你的角色卡上的先攻欄寫下你的敏捷調整值。
護甲等級Armor Class 。在沒有護甲和盾牌的情況下，你的基礎護甲等級是10加上你的敏捷調整值。如果你的起始裝備包含護甲或盾牌（或兩者皆有），使用第六章中的規則計算你的AC。職業特性可能會為你提供不同的AC計算方式。
攻擊Attacks 。在角色卡的武器&傷害戲法Weapons&Damage Cantrips部分寫下你的起始武器。你具有熟練的武器的攻擊檢定的加值計算方式如下，除非該武器的詞條另有說明：

近戰攻擊加值Melee attack bonus = 力量調整值 + 熟練加值

遠端攻擊加值Ranged attack bonus = 敏捷調整值 + 熟練加值

檢視第六章中你的武器的傷害和詞條。你使用武器進行攻擊時使用的屬性調整值也同樣加在傷害擲骰中。
施法Spellcasting 。使用以下公式，計算出並記錄你法術的豁免DC和你使用法術進行攻擊的加值：

法術豁免DC Spell save DC = 8 + 施法屬性調整值 + 熟練加值

法術攻擊加值Spell attack bonus = 施法屬性調整值 + 熟練加值

你的法術的施法屬性調整值取決於讓你獲得施展該法術能力的特性。
法術位、戲法和準備法術Slots、Cantrips，and Prepared Spells。如果你的職業給你施法或契約魔法特性，你的職業特性表中會顯示你可以使用的法術位數量、你已知的戲法數量和你可以準備的法術數量。選擇你的戲法和準備法術，並把它們和你的法術位數量一起記錄到你的角色卡中。

為你的角色取名 Name Your Character
為你的角色取一個名字，然後將它寫到你的角色卡上。名字可以隨你喜歡。它是一位先祖的名字嗎？它有某種宗教含義或者其他特別的意義嗎？還是說它就是你為自己選擇的名字？

建立最後一點細節 Create Final Details
當你建立完角色后，想想看你會為自己的角色新增的細節。這裡有一些你作為角色可能會問自己的內容：

  • 你的性別（代詞）是什麼？
  • 你會最關注或在乎什麼樣的人或人群？
  • 你最害怕最恐懼的東西是什麼？
  • 在你的冒險中，你在尋求什麼？那是知識、財富、榮耀、啟迪、正義、慈悲、力量還是其他的什麼東西？
