船隻數據卡Ship Stat Block

爲了支援進行船隻參與戰鬥，或者精確導航和麵對更多其他情況相關的冒險，以下部分提供了新的規則和一些列船舶數據以供使用。

基礎數據

船隻數據卡主要有三個部分：基礎數據，動作選項和船隻部件。船隻無法自己執行動作。離開的船員的操作，一艘船可能會隨波逐流，停靠在某處或者脫離控制。

體型 Size
大部分的船都是大型，巨型和超巨型。一艘船的體型大小取決於它的更長的一邊的長或寬，比如，一艘船有10尺長和20尺寬，它的體型取決於更長的寬也就是20尺寬，它是超巨型。

空間 Space
一艘船不會有內部空間除非它的資料板有特別說明。比如，一艘20尺長，10寬的船有一個20乘10尺的空間。一艘船無法移動進入一個無法容納它的小空間，如果它這麼做，它會撞擊，如同後面關於「撞擊」的描述。

容量 Capacity
一艘船的資料板會顯示它可以運載多少生物與貨物。生物包括操縱船的船員和乘客。乘客也包括抵禦登船者，攻擊怪物和敵船的士兵。

旅行里程 Travel Pace
旅行里程取決於這艘船在一個小時和一天之內可以走多遠。一艘船的移動力與它的部件相關且決定這艘船在一輪內可以移動多遠。

屬性值 Ability Scores
一艘船有六項屬性值（力量，敏捷，體質，智力，感知，魅力）和對應的調整值。一艘船的力量代表它的體型和重量。敏捷代表它的操作的流暢度。體質包含了它的耐久與建造質量。一艘船的智力，感知和魅力通常為0。如果這艘船的一項屬性為0，它在需要用到那個屬性的技能檢定和豁免自動失敗。

易傷，抗性和免疫 Vulnerabilities, Resistance,and lmmunities
一艘船的易傷，抗性和免疫取決於它的部件，除非它的資料板特別註明。船隻通常會免疫毒素和心靈傷害。他們通常還免疫以下狀態，目盲，魅惑，耳聾，力竭，恐慌，失能，麻痹，石化，中毒，倒地，震懾和昏迷。

動作 Actions

資料板的這一部分指出了這艘船能在它的回合做什麼，這些動作是特殊動作而非如同生物使用的動作。它甚至需要靠動作來移動，否則它無法移動。船隻的船長決定它採取什麼樣的動作。每個動作在每一回合之中只能被選擇一次。

部件 Components

一艘船由不同的部件組成。
船體Hull。一艘船的船體是它的基礎構架，其他的部件都安裝在船體上。
操縱Control。操縱部件被用來操縱船。
移動Move。移動部件是使船只能夠移動的要素。比如船帆和槳。
武器Weapon。一艘船在戰鬥中可以使用一到多個武器部件，而且每一個武器部件都分開使用。船隻數據卡可能會描述某些船隻部件的特殊規則。

護甲等級 Armor Class
每個部件都有護甲等級AC。它的AC反映在它的體型，製作材質以及任何用來提升它防禦力的鍍層和護甲。

生命值 Hit Points
一個部件會在生命值降到0時損毀或變的無法使用。如果一艘船所有的部件都被摧毀了，這艘船便被破壞了。船隻沒有生命骰。

傷害閾值 Damage Threshold
如果一個船隻部件有傷害閾值，傷害閾值會顯示在它的生命值後面。一個部件會免疫所有的傷害除非它所受到的那次傷害等於或大於它的傷害閾值，這會使其正常受到那次傷害。沒有等於或超過傷害閾值的傷害會被無視且不對部件造成傷害。
