半蛛神眷 Choldrith

半蛛神眷是直接侍奉羅絲的、巨大蜘蛛形生物。他們統治著半蛛人（在此書中）的聚落，並帶領他們參與對抗羅絲與敵人的紛爭。
當羅絲的信徒創造第一個半蛛人時，羅絲觀看著她的追隨者使用奧術魔法與惡魔力量與確保接受儀式者從中倖存所必須的神聖火花。作為對信徒的背叛帶來的復仇，蛛后扭曲了創造儀式，使他們有時創造的是半蛛神眷而不是半蛛人。
起初，這些信徒並沒有意識到這些他們稱為半蛛神眷的新的生物是羅絲對他們不滿的跡象。相反，他們十分高興。因為半蛛神眷可以產下能生出更多半蛛人（以及少部分神眷）的卵，並能在工作中引導半蛛人。但隨即，信徒們就意識到了他們的錯誤——半蛛神眷的身體和靈魂都屬於羅絲。半蛛神眷向半蛛人們低語它們對蛛后的崇拜和她們對卓爾的敵意，並帶領它們成功地進行了反抗。
半蛛神眷生來就與羅絲有一種神秘的聯繫，這讓它們具備一些神術魔法。它們也構成了大多數半蛛人聚落的統治階層。一個聚落可以支援大量半蛛神眷，擔任指揮官、祭司和檢察官等職位。半蛛神眷很少以一種危及整個聚落的方式相互對抗，但它們不斷地爭奪地位。聚落由一位君主統治，並由君王決定聚落成員執行哪些任務，，包括是否允許半蛛神眷產卵。有時，一個半蛛神眷統治者會從羅絲那裡收到一個幻象，激勵整個聚落進行一些宏大的，通常是暴力的行動。

半蛛神眷Choldrith
中型怪獸（牧師），普遍混亂邪惡
護甲等級：15（天生護甲）
生命值：66（12d8+12）
速度：30尺，攀爬30尺
力量12（+1） 敏捷16（+3） 體質12（+1）
智力11（+0） 感知14（+2） 魅力10（+0）
技能：運動+5，宗教+2，隱匿+5
感官：黑暗視覺60 尺，被動察覺12
語言：地底通用語
挑戰等級：3（700XP）熟練加值：+2

妖精血統Fey Ancestry。半蛛神眷進行對抗魅惑的豁免檢定時具有優勢，且魔法效應無法使半蛛人陷入睡眠。
蛛行Spider Climb。半蛛神眷可以攀爬在難以攀爬的表面上，包括倒吊在天花板上，且不需要進行屬性檢定。
日照敏感Sunlight Sensitivity。在日光照射下，半蛛神眷的攻擊檢定具有劣勢，其依靠視力的感知（察覺）檢定也具有劣勢。
蛛網感知 Web Sense。與蛛網接觸時，半蛛神眷可以知曉其他與該蛛網接觸生物的準確位置。
蛛網行者 Web Walker。半蛛神眷在蛛網上移動時無視蛛網造成的移動限制。
動作Actions
匕首Dagger。近戰或遠端武器攻擊：命中+5，觸及5尺或射程20/60尺，單一目標。命中：5（1d4+3）點穿刺傷害，外加10（3d6）點毒素傷害。
蛛網Web（充能 5~6）。遠端武器攻擊：命中+5，射程 30/60尺，單一大型或更小體型的生物。命中：該生物被蛛網束縛。受束縛生物可以用一個動作進行一次DC11的力量檢定，並在成功時以此掙脫蛛網。攻擊並摧毀蛛網也可以結束該效應（這些蛛網擁有AC10，5 點生命值，對火焰傷害易傷，並且免疫鈍擊、毒素和心靈傷害）。

隨意：神導術guidance，奇術thaumaturgy
每項1/日：災禍術bane，定身類人hold person
附贈動作Bonus Actions
靈體匕首Spectral Dagger（在短休或長休后恢復）。半蛛神眷在離自己60尺範圍內創造出一把浮空的虛幻武器。半蛛神眷可以對匕首5尺內的一個生物進行一次近戰法術攻擊（+4命中）。命中時，目標受到6（1d8+2）點力場傷害。
匕首將存在一分鐘。在後續回合中以一個附贈動作，半蛛神眷可以將匕首移動至多 20 尺，並再對匕首周圍5尺內的一個生物發動一次同樣的攻擊。
