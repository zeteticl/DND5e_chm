諾格弗羅特Nogvurot
人口:15270(70%人類，17%矮人，8%精靈，5%其他種族)
政府:Starosta Brumae Luddie在執法官的王冠衛隊Crownsguard的支援下處理城市事務。斯達洛斯塔Starosta屬下的貴族掌管地方行政。
防禦:許多王冠衛隊Crownsguard與一個被稱為為「最後神盾」（Last Aegis）的正義烙印士兵團Righteous Brand共同承擔保護職責。
商業:庫存充足的貿易站和旅館使大多數物資都很容易獲得，但很難找到稀有或異國情調的商品。
組織:有帝國認可的神明的寺廟，大多數企業都是在沒有公會監督的情況下獨立經營的。
諾格伍羅Nogvurot是從首都向東的榮耀大道Glory Run Road上的一個主要站點，接待著穿越帝國的旅行者，他們中的許多人需要補給，並喜歡購買當地製造的小飾品作為旅途的紀念品。在城市北部的褐巖山Dunrock Mountains山腳下的一個小礦場里有一些生意可做，但大多數諾格伍羅人靠手工藝、耕種或加入當地軍隊來維持生計。

健康的防禦A Healthy Defense
這個定居點最初是作為 Quannah Breach 附近的哨所而建立的，以監視從薩瓦里之森遊蕩的致命生物，該定居點後來發展成為正義烙印的一個相當大的軍事儲備。駐紮在城內的正義烙印團將成為抵禦任何前往首都的攻擊的防禦部隊，並授予他們「最後神盾」（Last Aegis）的稱號。該市近一半的人口是後備軍，在任何特定時間都有超過六千名士兵在場。無聊或焦躁不安的士兵爲了消遣和娛樂，將神盾區Aegis Block的兵營發展成為一個體育和競技遊戲的場所，通常會在結果上投入大量的硬幣。然而，在關於克靈人佔領Ashguard Garrison的報道和關於克靈人在骨髓谷Marrow Valley內部發生無法解釋的襲擊的謠言之間，神盾對此非常緊張。
政府Government
Starosta Brumae Luddie，一位正義烙印Righteous Brand的退役將軍，已經守護了這座城市十五年。布魯梅被強烈的責任感和對帝國的自豪感驅使，從不質疑國王的話。由於布魯梅被大多數人視為公平公正的 Starosta，一些人擔心他的高齡將很快結束他的任期，並讓不那麼謹慎的人物爭奪他的職位。
犯罪Crime
諾格伍羅Nogvurot 內強大的軍事力量將城市內的輕微犯罪降至最低。不幸的是，王冠衛隊Crownsguard經常挺起胸膛，試圖與正義烙印Righteous Brand的士兵單打獨鬥——一些當地的犯罪分子利用這種小規模的競爭來分散士兵的注意力，並繼續他們的犯罪行為不受阻礙。腐敗也成為一個問題，因為最後神盾Last Aegis的士兵利用他們在城市中的影響力，通過對進出 Shadycreek Run 和灰敗荒地Greying Wildlands其他犯罪熱點的走私視而不見來輕鬆賺錢。
地理Geography
這座長方形的城市分為東部的神盾區Aegis Block和西部地區，東部地區是大部分駐紮士兵，西部地區被分為兩個部分：北部的分散區Scatters和南部的小販街Peddler's Row。分散區Scatters 是更大的區域，擁有小酒館、簡陋的寺廟和大多數平民，他們住在許多搖搖欲墜的住所社區。相比之下，小販街Peddler's Row 是當地手工藝行會的繁華之家，也是榮耀大道 Glory Run Road 穿過城市的一個色彩繽紛但泥濘不堪的市場。

諾格伍羅的冒險Nogvurot Adventures
角色可以去諾格伍羅尋找城裡的某個人或加入正義烙印Righteous Brand。對於希望在與訓練有素的士兵的競技遊戲中測試自己勇氣的冒險者來說，這座城市也是一個好地方。
誰是叛徒（低階）Traitors In The Midst (Low Level)。 特爾瓦Telva Cornwrath（絕對中立，女性，人類間諜）接近角色們，並他聲稱為奧根信託Augen Trust工作。 她僱傭這些角色滲透到諾格伍羅 Nogvurot 的正義烙印Righteous Brand的行列，並發現哪些士兵正在接受 暗影幫 Shadycreek Run 的賄賂。 她想知道是哪個犯罪家族賄賂了每個腐敗的官員。她可能是她聲稱的那個人，或者她可能是萬象會Myriad的間諜，或者是暗流幫 Shadycreek Run 中尋求消除競爭的幫派之一。 在後一種情況下，她可能不希望角色留在那裡無所事事。
