暗流城Shadycreek
人口：14,770（56% 人類，15% 精靈，14% 矮人，15% 其他種族）
政府：四個有組織的犯罪家族和公民經營的灰商聯盟爭奪城市的控制權，而怨恨幫的僱傭維和部隊則試圖阻止徹底的混亂。
防禦：Shadycreek Run 的幫派們各自僱傭私人士兵和保鏢，而怨恨幫則保衛著整個城市。
商業：大多數普通商品都可以買到，稀有進口商品和被盜商品被秘密出售和拍賣。許多小酒館、旅館和罪惡場所都向公眾開放。
組織：大多數企業由部落或聯盟之一經營，即使聯盟是隱藏的。這個城市有分散的，簡陋的寺廟。經過懷南迪爾的北部前哨，一排分散的棚戶區和小屋將小路堵在一條小溪旁，通往荒地的樹木。
最初由殘酷的貧民窟領主經營的臨時貧困社會拼湊而成，Shadycreek Run 現在被稱為犯罪活動和野蠻人性的蜂巢，自由統治。墮落的文明渣滓將這裡視為獲得無數名聲和權力的機會。大多數辯論以鬥毆告終，而大多數鬥毆以流血而告終。
嚴酷、無人看守的環境孕育了堅韌的民族，儘管有肆虐的野獸從郊區捕食容易的獵物，但自豪的獵人和剝皮者群體已經崛起成為傑出的貿易商。
一些聰明的人物已經開始在 Run 中組織和文明生活，希望為收集Reprobates 帶來一點秩序和尊重，而 Run 的退伍軍人則努力讓事情變得不可預測，以暴力破壞這些嘗試。動盪的社會強調了這句古老的格言，「他們稱其為跑步是有原因的。」

危險的職業Dangerous Vocations
許多選擇住在Shadycreek Run的人這樣做是爲了避免帝國內部等待的迫害(或起訴)，而另一些人則喜歡在這片公開的道德靈活性的土地上享受殘酷的生活方式。這裡的職業從工匠、工匠到農民、僱傭兵和藝人都有。然而，該地區最賺錢的企業家是小偷、獵人和拾荒者，他們勇敢地面對致命的薩瓦里爾森林和莫拉斯米爾的廢墟。強壯的野獸能產出豐盛的肉和毛皮，而恢復的文物和丟失的魔法在遙遠的市場上能賣到相當高的價格。爭奪這些戰利品的鬥爭常常變得血腥，部落和灰商人聯盟為獲得這些貿易的控制權而戰鬥，同時也與烏托敦的人們競爭，後者希望收回他們的遺產和丟失的物品。

競爭關係Adversarial Rulership
目前，Shadycreek Run分為三個競爭派系：Greytrader聯盟、部落Tribes和仇恨幫派Grudge Gang。
格雷特工會（Greytrader Union）是一個由改革后的被驅逐者和罪犯組成的派別，他們希望與外部社會建立牢固的貿易關係，並通過使Shadycreek成為一個更安全、功能更強大的社區來幫助挽救Shadycreek。工會成員不遵守帝國統治，他們享受著無法無天的生活方式的自由，但他們也希望擺脫Shadycreek Run衆所周知的暴力遺產。由各行各業的捕獵者、獵人、建設者和工匠組成的格雷特工會希望成立自封的元帥來維持和平。他們的目標是增加平民的安全，將逃亡變成一個體面的貿易目的地，並最終減少部落和仇恨團伙的影響。
這些部落是敵對的犯罪家族的集合，他們在暴力內訌和內部權力鬥爭中失去了大部分統治地位之前，已經逃亡了近一個世紀。這些部落由馬爾登、烏托洛特、特雷班和賈根托斯家族組成，致力於控制城市中發生的大部分業務，並秘密相互勾結。雖然每個家庭中心通常由血緣或婚姻關係的親屬組成，但只要他們保持忠誠，他們的員工也被視為「家庭內部」。部落圍堵贓物，提供邪惡和淫穢的娛樂，並通過混亂、恐嚇和恐懼控制民眾。
懷恨在心的幫派是一群頑固的僱傭軍，他們充當力量，在逃亡期間強制執行當地利益。由德文達利軍隊的一隊逃兵建立，他們已經成長為一個忠於硬幣和彼此的戰士營。只要他們的生意不至於血腥到傷害無辜者或危及逃跑的安全，幫派成員將很樂意為格雷特工會或部落的任何家庭工作。因為其他派系依靠仇恨幫派來保護自己免受
在薩瓦里之森Savalirwood，其成員認為自己是政治棋局中的中立工具。該團伙的一些成員指派自己在一片沒有法律的土地上維持秩序，稱自己為任務者，並盡最大努力防止無辜者被城市的掠奪者壓榨。他們的首要任務是把他們的技能借給支付得最好的人，維護宜居的街道，防止薩瓦里之森和敦達里安帝國的危險進入城市。
政府Government
這座城市沒有中央政府，也沒有法律，所以它在很大程度上是由一種模糊的「生活和讓別人生活」的觀念維繫在一起的，偶爾也會被「拿我想要的」的情緒打破。如果暴力活動失控，咒怨幫的任務員就會介入，把所有參與者打成肉醬。粗心大意的人死在被雪覆蓋的小巷裡，身上的東西被剝光，被飢腸轆轆的狗吃掉，這並不罕見。逃亡部落的居民學會了提防他們的背部和小心地揮舞刀劍。
地理位置Geography
最初是沿著薩瓦里森林南部的林木線建造的一長串棚戶區和棚屋，現在已經變成了一條三英里長、彎曲的主街，被稱為三葉草巷，兩旁是小屋、建築物和泥坑。小巷南側的社區被稱為「南三葉草」（South Clover），是這座城市最貧困、維護最少的地區，以極度絕望的居民和更危險的生意而聞名。北部被稱為北三葉草，部分建在薩瓦里爾森林中，居住著富裕而有權勢的居民，包括大多數部落的家庭，他們住在各種豪宅和小堡壘里。
三葉草巷的西端被稱為Wickerskids，是食腐動物和農民的家園，他們竭盡全力在寒冷的北方貧瘠土地上為暗流邦Shadycreek Run的居民生產食物。三葉草巷的中心廣場被命名為三葉草廣場，標誌著城市商業、衝突和罪惡的中心。在廣場上，所有派系在光天化日之下混雜在一起，用一種放縱和過度的淫穢和喧鬧的生活方式來掩蓋他們隱藏的議程。四葉草巷的東端被稱為格拉夫貧民窟，住著一群農民和扒手，他們住在毒品窩點和豬圈裡。

暗影的冒險Shadycreek Run Adventures
寬鬆的法律和輕鬆賺錢的承諾將許多機會主義者吸引到這些危險的街道上。
一顆被盜的心（低階）A Stolen Heart (Low Level)。在 Trench Tavern，角色們加入了與當地怨恨幫的飲酒遊戲。無論輸贏，該團伙的一名礦石成員 Dratto 透露了他們與 Mardoon 家族的聯繫。 Dratto 提到他們需要一些人來竊取目前由 Trebain 家族持有的 Molaesmyr 遺物。角色可以將 Dratto 變成 Trebains 以獲得他們的青睞或同意協助盜竊。
如果角色同意幫助 Dratto，就會發生危險的入室盜竊。他們必須潛入 Wickerskids 外面的一個守衛棚屋，才發現這件遺物實際上是一個沉睡的小精靈。這個生物在角色面前醒來，當一個角色天性的一個神秘方面將小精靈從它的麻木中拉出來並將它的感情與角色聯繫在一起時。
國王的間諜（任何級別）King's Spy (Any Level)。 Nella Durlon（守序中立，非女性，人類間諜）作為帝國間諜居住在 Shadycreek Run。他們收集有關該定居點防禦的資訊並帶回給 Oliver Schrieber（參見第 2 章中的「Dwendalian 帝國」）。角色們從僱傭他們尋找間諜的派系中得知了 Nelia 的工作。根據陣營的不同，角色必須要麼護送內拉安全離開定居點，以便向奧利弗報告，要麼要求殺死內拉以泄露定居點的秘密。無論哪種方式，找到內拉都不是一件容易的事。間諜精通他們的手藝，並與逃跑的罪犯和惡棍混在一起。
